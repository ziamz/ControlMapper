About ControlMapper

Problem Statement: There are many scenarios where a Governance, Risk and Compliance practictioner needs to map a control against a set of controls. For example, mapping a regulatory requirement against an existing control library. This often involves identifying similar controls within the control library, and if applicable, modify the existing controls in the control library to meet the requirements.

Objective: ControlMapper is designed to help users map Custom Controls to an existing Control Library. It achieves this by looking for the most similar combinations using both keyword (word overlap) and semantic (words with similar meaning) algorithms. The tool does not map the controls, rather it helps users to identify similar controls and users are the ones to do the mapping.

Key Features

  • Similarity Scoring: Combines keyword matching and semantic meaning matching to find the most relevant controls.
  • Adjustable Weights: Adjust the % weightage between Keyword and Semantic scores.
  • Local Semantic Matching: Semantic matching models are downloaded and run entirely locally within your browser!
  • Generative AI Gap Analysis: Only if enabled, note this will send data to LLMs - Uses Large Language Models (LLMs) to automatically detect and summarize gaps between your mapped controls.

How to Use

  1. Upload Data: Navigate to the Upload tab. Upload your Control Library and Custom Controls data (Excel format). Select which columns contain the IDs and Descriptions. The "Descriptions" column is used for similarity matching.
  2. Local Semantic Matching The tool automatically downloads a local Transformer model for semantics comparison. Initial model download will require a bit of time (2 minutes). Model used: all-MiniLM-L6-v2. If Github source is blocked, download the model separately from Github repo and save it in the same folder as the tool.
  3. Map Controls: Navigate to the Mapping tab to view similar controls. You can customize the number of top matches to manually review. Once you have decided on the control to map, click on Map.
  4. Configure LLM Gap Analysis (Optional): If you want LLM to perform Gap Analysis, you need to input an API key. Go to the Settings tab, enable AI Gap Analysis, select your provider, and input your API key (e.g. Gemini or OpenAI). You can also customize the AI prompts here. Once done, you can use the Gap Analysis feature (Click Trigger LLM Analysis) to compare the 2 controls. If you map multiple controls, you can use the Group Analysis feature to compare these multiple controls against the control.
  5. Export: You can click the Download Results button to export everything to Excel at any point in the process. You can choose to do the AI Analysis in the Tool then do the Mapping in Excel if it is easier.
